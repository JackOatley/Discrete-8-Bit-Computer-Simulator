Clock Speed: 8

Reference

NOP - No operation, continues to next instruction.

HLT - Halt, but no catching fire.

SEA [value] - Set A, sets the value of Register A to the given value.

SEB [value] - Set B, sets the value of Register B to the given value.

LDA [address] - Load A, loads the value at the address into Register A.

STA [address] - Store A, stores the value of Register A to the given memory address.

STB [address] - Store B, stores the value of Register B to the given memory address.

ADD [address] - Add, add the value at the given memory address to the value in Register A.

SUB [address] - Subtract, subtract the value at the given memory address from the value in Register A.

ADR - Add Directly, adds the values in registers A and B together.

SBR - Subtract Directly, subtracts the value in register B from the value in register A.

JMP [address] - Jump, jumps to the given memory address.

JLT [address] - Jump If Less Than, jumps to the given memory address if Register A value < Register B value.

JIZ [address] - Jump If Zero, jumps to the given memory address if Register A value is 0.

SWP - Swap, swaps the values of registers A and B.

OUT - Output, puts the value of Register A into the Display Register.

Notes

This is a very early version of the intended program, and has only had a couple days where I worked on it. As such the code has not been refactored, and a lot of features are missing. Future features will include: labels (will make dealing with memory addresses a lot easier), instruction cycle simulated (so you can see the control word and it's effects), comments, syntax highlighting, error reporting, logs of changes in hardware per run, and loads more stuff!
